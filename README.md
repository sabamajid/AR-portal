# AR-Portal
Augemented Reality Portal

Developed an Augmented Reality app for Android and iOS devices using Unity, and AR Foundation.In Portal, the user can go through one door and can enter into the Virtual world.



Unity 2019.4
AR Foundation
Unity Assest Store
